# OrangeHRM-Automation-Testing

This project automates the OrangeHRM web application using Selenium WebDriver with Java.

## Tools & Technologies
- Java
- Selenium WebDriver
- TestNG
- Cucumber BDD
- Maven

## Framework Used
- Page Object Model (POM)
- Hybrid Automation Framework

## Modules Automated
- Login Module
- Add Employee
- Edit Skills
- Delete Job Title
- Membership Module

## Testing Concepts
- SDLC
- STLC
- Smoke Testing
- Regression Testing
- Functional Testing
